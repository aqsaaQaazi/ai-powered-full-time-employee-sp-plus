# SKILL_CreateFolders

## Purpose
Creates a basic folder structure for organizing tasks and projects in an Obsidian vault.

## Prompt Logic
Use this skill to create the essential folder structure consisting of Inbox, Needs_Action, and Done directories for organizing tasks and projects.

## Instructions
1. Create the following directories in the current location:
   - Inbox (for new tasks and ideas)
   - Needs_Action (for tasks requiring attention)
   - Done (for completed tasks)
2. Verify that all directories were created successfully
3. Optionally list the directories to confirm creation

## Expected Outcome
Three directories will be created: Inbox, Needs_Action, and Done, allowing for proper organization of tasks and projects in the vault.
