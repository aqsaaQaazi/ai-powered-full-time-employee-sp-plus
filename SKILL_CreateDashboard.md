# SKILL_CreateDashboard

## Purpose
Creates a dashboard file with real-time summary of bank balance, pending messages, and active business projects.

## Prompt Logic
Use this skill to create or update the AI Employee Dashboard with current information. Replace placeholder values with actual data as needed.

## Template
```
---
title: AI Employee Dashboard
last_updated: {current date}
---
## Real-Time Summary
- Bank Balance: Placeholder
- Pending Messages: 0
- Active Projects: None

## Recent Activity
- No activity yet.

This is the real-time summary of bank balance, pending messages, and active business projects.
```

## Instructions
1. Replace `{current date}` with today's date in YYYY-MM-DD format
2. Update placeholder values with real data as available
3. Modify Recent Activity section with actual activities
4. Save the file as Dashboard.md in the root directory
