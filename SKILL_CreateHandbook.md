# SKILL_CreateHandbook

## Purpose
Creates a company handbook with rules of engagement for AI employee operations.

## Prompt Logic
Use this skill to create or update the Company Handbook with current rules of engagement. This document sets guidelines for the AI employee's behavior and decision-making processes.

## Template
```
---
title: Company Handbook
last_updated: {current date}
---
## Rules of Engagement
- Always be polite in communications.
- Flag any payment over $500 for approval.
- Prioritize urgent tasks.

This contains rules like "Always be polite on WhatsApp," "Flag any payment over $500 for my approval".
```

## Instructions
1. Replace `{current date}` with today's date in YYYY-MM-DD format
2. Add or modify rules of engagement as needed
3. Customize the content to reflect current company policies
4. Save the file as Company_Handbook.md in the root directory
