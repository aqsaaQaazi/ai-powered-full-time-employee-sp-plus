# Master Skills List

This document lists all available Agent Skills for the AI Employee vault system.

## Available Skills

### SKILL_CreateFolders
- **Description**: Creates basic folder structure (Inbox, Needs_Action, Done)
- **Purpose**: Sets up initial organizational structure for the vault

### SKILL_CreateDashboard
- **Description**: Creates/updates the AI Employee Dashboard with real-time summary
- **Purpose**: Provides a centralized view of bank balance, pending messages, and active projects

### SKILL_CreateHandbook
- **Description**: Creates company handbook with rules of engagement
- **Purpose**: Establishes guidelines for AI employee behavior and decision-making

### SKILL_VaultIO
- **Description**: Handles read/write operations in the vault
- **Purpose**: Tests and performs file operations, verifies file accessibility

### SKILL_GenerateWatcher
- **Description**: Generates filesystem watcher scripts
- **Purpose**: Creates Python scripts that monitor drop folders and process files automatically

## Additional Skills

### SKILL_TestWatcher
- **Description**: Tests the File System Watcher functionality
- **Purpose**: Verifies that the filesystem watcher operates correctly by testing file movement and metadata creation
